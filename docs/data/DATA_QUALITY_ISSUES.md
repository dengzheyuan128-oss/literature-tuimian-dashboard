## 🔧 修复指南

### 如何获取正确的推免通知链接

#### 方法一：访问研究生院官网

1. **访问研究生院**
   - 通常为：`https://grs.xxx.edu.cn` 或 `https://yjsy.xxx.edu.cn`
   - 例如：北京大学研究生院 `https://grs.pku.edu.cn`

2. **查找招生信息栏目**
   - "通知公告"
   - "招生信息"
   - "推免招生"
   - "硕士招生"

3. **搜索关键词**
   - "推免"、"保研"、"接收推荐免试"
   - "硕士研究生"
   - "2024年"、"2025年"

#### 方法二：访问文学院官网

1. **访问文学院/中文系官网**
   - 通常为：`https://wxy.xxx.edu.cn` 或 `https://chinese.xxx.edu.cn`
   - 例如：北京大学中文系 `https://chinese.pku.edu.cn`

2. **查找招生信息栏目**
   - "通知公告"
   - "招生信息"
   - "研究生招生"

3. **确认链接有效性**
   - 链接应直接指向具体的招生通知文件或页面
   - 通知应包含报名时间、材料要求、考核方式等详细信息

### 链接质量标准

#### ✅ 正确示例

```
https://chinese.pku.edu.cn/jyjx/zsxx/5e733db8e74e4391b4b8fd430540dea3.htm
```
- 包含具体的通知ID或文件名
- 指向具体的通知页面
- 包含详细的招生信息

#### ❌ 错误示例

```
https://www.pku.edu.cn/
https://chinese.pku.edu.cn/
http://www.lianpp.com/pku/chinese/notice.htm
```
- 学校首页
- 学院首页
- 第三方平台链接

## 📝 修复流程

### 1. 选择要修复的院校

从上述清单中选择一所院校

### 2. 搜索推免通知

使用搜索引擎搜索：
```
[院校名称] 文学院 推免 2024
[院校名称] 中文系 保研 2024
[院校名称] 研究生院 接收推免 2024
```

### 3. 验证链接有效性

- 确认链接可以正常访问
- 确认通知内容包含推免招生信息
- 确认通知时间为2024年或2025年

### 4. 更新数据文件

编辑 `client/src/data/universities.json`，更新对应院校的 `url` 字段：

```json
{
  "id": 1,
  "name": "北京大学",
  "url": "https://chinese.pku.edu.cn/jyjx/zsxx/正确的推免通知链接.htm"
}
```

### 5. 运行数据检查

```bash
node scripts/check-data-quality.js
```

### 6. 提交更新

```bash
git add client/src/data/universities.json
git commit -m "fix: 更新[院校名称]推免通知链接"
git push
```
