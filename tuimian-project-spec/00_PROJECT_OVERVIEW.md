# 项目总览

## 项目定位

**钝学推免指南** - 985/211院校文学类推免信息聚合平台

## 核心目标

为文学类推免学生提供：
- 全面的院校推免信息（62所985/211院校）
- 结构化的院校分级（五梯队体系）
- 可追溯的官方通知链接
- 数据质量保障机制

## 当前阶段

**阶段1**: 院校名录数据补全与质量提升

### 当前进度
- 院校总数：62所
- 数据质量：
  - A级链接：0所 → 目标13所（第一批）
  - D级链接：2所 → 目标0所
  - 整体健康度：96.8% → 目标100%

### 停损线指标
1. D级链接 = 0
2. 整体健康度 ≥ 90%
3. 第一梯队A级占比 = 100%

## 技术栈

- 前端：React 19 + Tailwind 4 + shadcn/ui
- 数据：JSON静态数据
- 部署：Manus托管

## 协作方式

- 版本控制：GitHub
- 文档驱动：tuimian-project-spec/
- 提交规范：强制commit message模板
- 数据更新：Excel驱动 + 人工确认

## 关键原则

1. **文档优先于对话**：所有规则必须文档化
2. **提交优先于描述**：未提交=不存在
3. **冻结优先于优化**：先定死规则，再执行
4. **单线程原则**：一次只做一件事
